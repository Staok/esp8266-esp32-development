使用在线安装：
	注：这样安装是 在 本地 安装一个 IDE，用 vscode 安装的方式是下载一套 其的 esp-idf 插件，两种都可以
	1、关闭杀毒软件，用管理员方式打开 esp-idf-tools-setup-online-2.16.exe。
	2、选择路径，推荐只改 盘符而不要动其它路径名，比如 只 改 c 盘 到 d盘；然后选择安装组件，在选择 ”完全安装“ 的基础上 再勾选 esp-idf-ide（里面包含 快捷方式 和 JDK）！
	3、确保网络良好，不要在下载的时候中断，否则 可以点 retry，或者重新打开 选择 esp-idf  解压的目录；安装。

使用 shell 编译工程：
	使用 idf.py 来编译、下载程序，idf.py 封装了 CMake 命令，也可以用 make 命令来操作，但推荐 idf.py
	
	打开 ESP-IDF 4.4 PowerShell

	首先要 shell 中进入工程目录
	
	如果是首次：
	idf.py set-target esp32s3
		选择目标 ic
		也可以直接将“目标”配置为环境变量：https://docs.espressif.com/projects/esp-idf/zh_CN/latest/esp32s3/api-guides/tools/idf-py.html#selecting-idf-target
	idf.py menuconfig
		进入 menuconfig 可以调整 目标 ic 的 flash 大小、串口端口号等等，还有 Wi-Fi 网络名称、密码和处理器速度等等，其它选项基本不用动。
	
	之后的编译程序：
	idf.py build
	
	下载程序：
	idf.py flash
		需要首先在 menuconfig 中指定好了端口（串口）

	显示编译后固件大小：
	idf.py size
	
	清理：
	idf.py clean
	
	全清：
	idf.py fullclean

	查看更多 idf.py 命令
	idf.py --help

	更多 idf.py 命令：
	https://docs.espressif.com/projects/esp-idf/zh_CN/latest/esp32s3/api-guides/tools/idf-py.html

	获取更多信息
	idf.py menuconfig --help






Windows 环境下 ESP-IDF 编译比较慢如何优化？
	CHIP: ESP32
	请将 ESP-IDF 源码目录以及编译器目录 .espressif 添加到杀毒软件的排除项。
	或者不用这么麻烦，直接关掉杀毒软件（360、电脑管家之类的）即可大幅提高编译速度
