# esp8266-esp32-development
收集关于 esp32 和 esp8266 主流的简便的开发教材、工具和例程，方便快速入门或者快速拾起。相关的 开发板 和 模组 可以看仓库内 “模组选型” 文件夹，或者到淘宝等地方搜索。一个仓库打包了 ESP MCU 相关的东西，仓库较大（200MB），**推荐压缩包方式直接下载 而不要使用 git clone 仓库**。
